= my_savage_beast Gem
This is a modified version of the savage beast plugin, updated to work with Rails 3.1 - to the extent that it works with our project - http://www.seek4science.org


More information about the original Plugin please see  https://github.com/wbharding/savage-beast

== Install
Put this line into Gemfile:
  gem 'my_savage_beast', :git => 'git://github.com/SysMO-DB/my_savage_beast'
